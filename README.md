# Installation

### For frontend

```
    git clone https://github.com/Groot-2001/Brained_Assignment.git
    cd frontend
    npm run dev
```

### For Backend

```
    cd backend
    npm run server
```

### Env Setup (In Backend)

1. Create .env file in backend folder.

```env
    DATABASE_URI = "Add your database uri"
```
